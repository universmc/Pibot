Comment utiliser Bootstrap

Bootstrap est un framework CSS et JavaScript open source pour le développement web rapide et facile.

Grid System

Bootstrap utilise un système de grille pour organiser le contenu de votre page web.

Components

Bootstrap offre une variété de composants prêts à l'emploi, tels que les boutons, les formulaires et les modaux.

Utilities

Bootstrap fournit de nombreuses utilitaires CSS pour faciliter le style de votre contenu.
